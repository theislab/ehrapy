ehrapy¶

Features¶

  • Exploratory analysis of Electronic Health Records

  • Quality control & preprocessing

  • Clustering & trajectory inference

  • Visualization & Exploration

Installation¶

You can install ehrapy via pip from PyPI:

$ pip install ehrapy
